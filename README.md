# LangGraph

This repository contains my practice work while learning **LangGraph** and **LangSmith**.

It includes examples of:

- Sequential workflows
- Parallel workflows
- Conditional workflows
- Iterative workflows
- Conversation persistence
- LangSmith tracing and observability

The purpose of this repository is to understand how agentic workflows are built, executed, monitored, and debugged using LangGraph and LangSmith.
